Exercise 4

USB Serial Instructions

Required material. Arduino Uno, Arduino multifunctional shield.
Worked instructions
Exercise 4.1: Hello world!
  1. Hello world! At the start up, print the sentence "Hello world!" on the console.
  2. Now, get the same result, but first print the word "Hello " and then print "world!" in two separate instructions. The resulting text must be shown on the same line (as before). Hint: Use print and println instructions.
Exercise 4.2: Loopback
  1. Serial loopback Create a program that sends over the serial port data received on it. Hint: Use the 'read string' instruction.
Exercise 4.3: Parsing numbers
  1. Parsing integer numbers Modify the previous program so that it expects an integer number . Read it and print on the console the successor. Hint: Use the 'read integer' instruction.
  2. Parsing float numbers Modify the previous program so that it expects a float number . Read it and print on the console its negate value. Hint: Use the 'read float' instruction.
Exercise 4.4: Plotting signals
  1. Plot a signal. Using Arduino's plotter, plot the analog value of the potentiometer (connected to pin A0). Hint: Use the 'plot data' instruction.
