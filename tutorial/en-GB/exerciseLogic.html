Logic

Learn how to work with logic instructions with Facilino, including Boolean and bitwise operations.

Required material.
  • Arduino Uno (and compatible) or Wemos D1R2/D1R32.
  • Arduino multisensor shield.

Exercises
  1. AND and OR Operations.. Implement binary operations using the button switches (connected to pins D2 and D3) and the red and blue LEDs (connected to pin D12 and D13). The aim is to implement the following logic operations: RED=¬(SW1∧SW2) (this is equivalent to AND Operation), BLUE=¬(SW1∨SW2) (this is equivalent to OR operation).
  2. Even or Odd (Bitwise And). Turn on LED D12 whenever we introduce an even number on the USB serial console. Turn on LED D13 when it's odd. Hint: Use a bitwise 'and' to mask the number with '1'. If the result is greater than zero then the number is odd.
  3. Comparison. Read the analog input A0 corresponding to the potentiometer position. Divide the analog read range into 4 sub-ranges and turn on LEDs D12 and D13 based on the binary codification of each range. Hint: If the number is between 0 and 1023, then both LEDs must be off, if the value is between 1024 and 2047 then LED D12 must be on and D13 off, if the value is between 2048 and 3071 then LED D13 must be on and D12 off. Otherwise, both LEDs must be on.
