USB Serial Instructions

Learn how to print to the console and read from the console with the USB serial interface.

Required material.
  • Arduino Uno (and compatible) or Wemos D1R2/D1R32.
  • Arduino multisensor shield.

Exercises
  1. Hello world! At the start up, print the sentence "Hello world!" on the console.
  2. Serial loopback Create a program that sends over the serial port data received on it (loopback). Hint: Use the 'read string' instruction.
  3. Parsing integer numbers Modify the previous program so that it expects an integer number . Read it and print on the console the successor (add one to the read number). Hint: Use the 'read integer' instruction.
  4. Parsing float numbers Modify the previous program so that it expects a float number . Read it and print on the console its negate value. Hint: Use the 'read float' instruction.
  5. Plot a signal. Using Arduino's plotter, plot the analog value of the potentiometer (connected to pin A0). Hint: Use the 'plot data' instruction.
