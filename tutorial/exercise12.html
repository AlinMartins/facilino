Exercise 12

128x32 OLED Display Instructions

Required material. Arduino Uno, 0.91'' 128x32 OLED Display.
Preliminaries
OLED Layout. The 0.91'' OLED Display is a 128x32 black/white display. In order to create graphic user-interfaces, a layout proposal is proposed, which contains separated regions for the corners, sides, header, main and footer sections. Each of them have a different size and top-left corner position as indicated on the following table:
What can I do? You can draw images of predefined size on each of these regions or write some text or (single) characters.
Worked instructions
Exercise 12.1: Printing text
  1. Print some text using the 'Set Text' instruction. You can choose the text position and size to your convinience. Hint: Make sure that the text position and size fits properly on the screen.
  2. Print some text on the header, main and footer sections that is meaninful for you. For instance, you can write "OLED 128x32 demo" in the header section, "Facilino" in the main section and "Cool! Isn't?" in the footer section. Hint: Take into account that the maximum number of characters as well as the text size is different for the header, main and footer sections. Header and footer text size is 1, which means is small an you should print something that it is not that important. Their maximum number of characters is 18. On the other hand, the main section uses a font of size 2, which makes only 8 characters possible.
  3. Now, modify the previous code so that you introduce a longer text in any of the sections, say for instance in the footer section. Replace your footer text with "https://roboticafacil.es". As you can see, the text won't fit, so the automatic behaviour is to horizontally scroll the text to show the full message. Please, check that!
Exercise 12.2: Drawing icons and images
  1. Draw 'Facilino's logo'. You can import the logo at the following URL https://roboticafacil.es/facilino/blockly/img/facilino40x32.png Draw the logo in the middle of the screen. Hint: The logo's width is 40, thus, (128-40)/2=44, so the X coordinate must be 44.
  2. Move 'Facilino's logo'. Now, make the logo move to the left and right continuously. Hint: Following the previous explanation about the logo position, now the logo must move from the x-coordinate position 0 to 87.
  3. Draw predefined icons and images. Now, use the set of predefined 8x8 and 16x16 icons and 96x16 images. The aim is to draw four of 8x8 icons on each of the corners, two of the 16x16 icons on the sides and the 96x16 image on the main area. Choose whatever you prefer. Hint: Use the layout instructions to properly place the icons and images.
Exercise 12.3: Drawing geometric objects
  1. Draw lines. Generate 9 lines that connect the top-left corner and a point on the right side of the screen and 32 lines that connect the top-left corner and the bottom side of the screen. The points on the right side are between the top-right corner and the bottom-right corner with a separation of 4 pixels. Similarly, the points on the bottom side of the screen are between the bottom-right corner and bottom-left corner. Hint: In a for loop, draw 9 lines from (0,0) to (128,0), (128,4), (128,8) .... (128,32). After that, in another for loop, draw 32 lines from (0,0) to (128,32), (124,32), (120,32) .... (0,32). Add a delay of 200ms between each line.
  2. Draw rectangles. Generate 5 rectangles on each direction of width 25 and height 6. The rectangles must be alternatively be filled or empty. Hint: Using two nested for loops, one for the horizontal direction and another for the vertical direction, draw rectangles of width 25 and height 6. The top-left corner to draw each rectangle must be (i*25,j*6), being i the iterator for the horizontal loop and j the iterator for the vertical loop (i=0...4) and (j=0...4). To alternatively generate filled rectangles, use a boolean variable that on each iteration negates its own value. If the variable is true, then use the fill rectangle option, otherwise, uncheck it.
  3. Draw triangles. Generate a pattern of 4x4 right triangles with long side of 32 pixels (horizontally) and a short side of 8 (vertically). Hint: Using two nested for loops with i=0...3 and j=0...3, use i*32 and j*8 for vertex0, i*32 and (j+1)*8 for vertex1 and (i+1)*32 and j*8 for vertex2.
  4. Draw circles. Draw 5 circles centered in the middle of the display with the same pattern as the Olympic Games symbol. Hint: If you do the maths, the centers of the circles are in (37,11), (63,11), (89,11), (50,20) and (76,20) with a radius of 10 pixels.
