Preliminaries
The rice and the chessboard story. There's a famous legend about the origin of chess that goes like this. When the inventor of the game showed it to the emperor of India, the emperor was so impressed by the new game, that he said to the man "Name your reward!"
"Oh emperor, my wishes are simple. I only wish for this. Give me one grain of rice for the first square of the chessboard, two grains for the next square, four for the next, eight for the next and so on for all 64 squares, with each square having double the number of grains as the square before."
The emperor agreed, amazed that the man had asked for such a small reward - or so he thought. After a week, his treasurer came back and informed him that the reward would add up to an astronomical sum, far greater than all the rice that could conceivably be produced in many many centuries!

Worked instructions
Exercise 8.1: The rice and the chessboard story
  1. How many rice grains in total? Compute the requested number of grains. Use different number representations to see which can represent the number. Show on the console the total amount computed when using bytes, integers, long integers and floats.
Exercise 8.2: Sum of two strings
  1. Summing integer strings. Declar integer variables 'num1' and 'num2' and assign to then the value after converting a string with numbers. Sum the numbers and convert them back into a string to show the result on the console. Check if it works!
  2. Now, implement a function that accepts two integer numbers (as string type variables) and returns the sum of the variables in another string type variable. Check if it works!
  3. Now, modify the previous function so that it accepts float numbers (as string variables), instead of integers. Check if it works too!
