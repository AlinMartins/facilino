Exercise 9

Math Instructions

Required material. Arduino Uno, Arduino multifunctional shield.
Worked instructions
Exercise 9.1: Binary digits
  1. Binary digits. Compute the number of digits required to represent an integer number in a binary representation. Implement a function that accepts as input argument an integer number and returns the number of digits. Use, for instance number '123'.Hint: In a while loop, check if the number is greater or equal than the base 2 powers (starting from 2). Each time that this is true, a digits counter must be increased (initialized at 1).
  2. Random numbers. Now, check how this function works for 30 randomly generated number between 0 and 199.
Exercise 9.2: Minimum and maximum values
  1. Minimum and maximum. Using the potentiometer (connected to pin A0) measurements, compute the minimum and maximum values in volts (the value must be comprised between 0 and 5 volts). At initialisation, turn on red LED (connected to pin D12) and perform the maximum and minimun computations until the switch SW1 (connected to pin D2) is pushed. Once pushed, turn off the red LED and show on the console the minimum and maximum values of all registered measurements. Hint: In a while loop, read the analog measurement in A0 and convert it into a number between 0 and 5 (using the map instruction). Use the min and max instructions to compute the corresponding minimum and maximum and save the results in two different variables (the variable storing the minimum value must be initialised with 5, while the variable for the maximum must be initialised with 0).
Exercise 9.3: Advanced math functions
  1. Square root. Compute the square root of numbers from 1 to 100 an show the result on the console. Check if the operations are correct.
  2. Inverse functions. Log and exponential functions are inverse functions, or so they say.... Can you check it? Compute the log of numbers from 1 to 100 and then compute the exponential function of the resulting number.
Exercise 9.4: Root Mean Square (RMS)
  1. RMS of a sinusoidal signal. The RMS of a sinusoidal signal is √2A/2, being A the amplitude of the sinusoidal signal (no matter the frequency). Create a function to compute of a sinusoidal signal with amplitude A=√2 (so the RMS must be 1). The frequency of the sinusoidal can be set, for instance to 0.1Hz, offset 0 and phase 0º. Hint: In a while loop, evaluate a sinusoidal wave using the corresponding math instruction. You need to measure the time from the start in microseconds 't0' before executing the while loop and provide to the sinousoidal wave instruction the time difference between the current time 't' and 't0'. Compute the RMS using its discrete implementation RMS=√((1/n)∑s(t)), being n the number of points and s(t)=A*sin(2πft). You can include a small delay, say of 0.1ms, on each new computation of the sinusoidal value (in the while loop).
