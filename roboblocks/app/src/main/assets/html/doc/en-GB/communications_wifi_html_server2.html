HTTP Server (WiFi ESP8266)

Instruction

Description

Defines an HTTP server using ESP8266 processors to publish HTML code.

Requirements

Facilino's License.

This instruction requires having configured external URLs for the board management in the preferences of Arduino IDE:

Subsequently, the ESP8266 card must be installed on the Arduino IDE Board Manager:

Inputs

  • SSID: Network Name (SSID).
  • Password: Password to access the network.
  • Port: Server port (default 80).
  • HTML: HTML document to publish.

Example

This example uses the ESP8266 processor to create a web that displays the message "Hello world!"
