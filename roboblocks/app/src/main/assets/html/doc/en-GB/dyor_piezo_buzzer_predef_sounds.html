Description

Generates a set of predefined sounds useful for many applications using a passive buzzer.

Requirements

Inputs

  • Pin: Digital pin where the buzzer is connected.

Dropdown lists

Example

This example the 'connection' sound is reproduced when the button is pressed and 'disconnection' sound when released.
