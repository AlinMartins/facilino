About Facilino

Facilino is a block programming tool, based on Blockly (Google), to program Arduino. It allows you, in a very easy way to implement high-level concepts. Facilino cn be used in a wide variety of projects because contains blocks for many kind of devices, with advanced features, simple to use, particularly with kids. Facilino must be seen as a code generation tool that will boost-up your programming skills.
