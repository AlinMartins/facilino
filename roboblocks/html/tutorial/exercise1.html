Exercise 1

Identification of Arduino Uno's board main components

Required material. Arduino Uno board, multimeter and male-male DuPont cables
Exercise 1.1: Power and Reset pins
  1. USB power-up. Power-up Arduino Uno's board using the USB cable to a computer. Now, meassure the voltage, using a multimeter the at pins 5V and 3.3V. How many of these pins you can find? Rembember to use the voltage mode in the multimeter to avoid short-circuiting the board.
  2. Battery power-up. Now, repeat the measurements, but using an external battery of 9V. Are the values the same?
  3. ON LED. Indentify the power-on LED on Arduino's board.
  4. 5V and GND pins. With the power supply off, use the multimeter to test the connectivity between 5V and GND pins. Check if you can hear a beep when testing pins of the same type, e.g. 5V<->5V or GND<->GND.
  5. Reset pin. Meassure the voltage at the RES pin (reset). Does this voltage change when pressing the RESET button? 
Exercise 1.2: Microcontroller and clock
  1. MCU. Locate Arduino Uno's MCU (Microcontroller Unit) IC (Integrated Circuit).
  2. Clock. Locate MCU's clock. It should be next to it and with a visible number 16.000 on it.
  3. MCU's pins. The MCU IC has a marked corner with a dot. Find out why is this dot used for.
Exercise 1.3: Digital and analog pins
  1. Digital pins. Count the number of available digital pins in Arduino Uno's board.
  2. Analog input pins. Count the number of analog input pins in Arduino Uno's board.
  3. Analog output pins (PWM). PWM pins are marked with the ~ symbol. How many of them you can find?
  4. Rx and Tx pins. Rx and Tx pins are used for serial transmistion over the USB cable. These pins are special and shouldn't be used in your project. Can you identify those pins?
