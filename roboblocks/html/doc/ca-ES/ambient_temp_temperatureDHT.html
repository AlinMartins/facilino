Llegeix sensor temperatura DHT11/DHT21/DHT22

Instrucció

Descripció

Retorna la temperatura en graus centígrads del sensor d'humitat DHT11/DHT21/DHT22

Desplegables

  • Sensor: Model del sensor utilitzat DHT11/DHT21/DHT22.

Entrades

  • Pin: Pin a què està connectat el sensor.

Eixida

Temperatura retornada pel sensor.

En aquest exemple es mostra per consola la temperatura llegida amb el sensor DHT11.
