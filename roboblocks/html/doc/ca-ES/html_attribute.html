Atribut d'etiqueta HTML

Instrucció

Descripció

Crea un atribut per a una etiqueta HTML.

Desplegables

  • Atribut: Tipus d'atribut a crear.
  • Codi: Codi per a l'atribut.

Eixida

Codi HTML per inserir en els atributs d'una etiqueta HTML.

Exemple

En aquest exemple s'utilitza el processador ESP8266 (integrat) per crear una web que mostra un títol en roig i el text amb format "Hola món!"

Es perdran tots els canvis!
