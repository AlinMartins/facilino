Envia Octetos I2C

Instruccions

Descripció

Esta instrucció s'usa per un maestro del bus I2C per a enviar dades a un esclavo I2C en la dirección indicada. La instrucció utilitza la variable proporcionada per a determinar la cantidad total de bytes a recibir (dependiendo del tamany del vector i del tipus de variable).

Entrades

  • Variable: Variable vector per a enviar dades.

Camps

  • Dirección: Dirección del esclavo I2C. El valor màxim de la dirección es 127.

Exemple

Aquest exemple mostra les capacidadades de la comunicación I2C. Representa una red distribuida de MCUs. Una d'ellas actua com a maestro que solicita información de la lecturad'un sensor de infrarrojos de dos esclavos independientes (en les direcciones 20 i 21). Els dades recibidos se adaptan a l'escala de 0 a 180 per a ser transmitidos a un esclavo que realiza les funciones de controlador de servomotores (en la dirección 10) i que es el encargado de mover els servos.
