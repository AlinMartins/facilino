Generar so (avançat)

Instrucció

Descripció

Genera un so amb la freqüència i durada especificats.

Entrades

  • Pin: Pin a què està connectat el brunzidor.
  • To: Freqüència a emetre (en Hertzios).
  • Durada: Durada del so (en milisegons).

Exemple

En aquest exemple, s'emet un so molt greu i un altre molt agut de diferents durades.
