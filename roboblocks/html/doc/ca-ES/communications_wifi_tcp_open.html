Obre connexió Client TCP / IP (ESP8266)

Instrucció

Descripció

Estableix una connexió TCP / IP amb un servidor com a client TCP / IP. Requereix de la IP del servidor i el port a comunicar i seleccionar el mode (WiFi integrat o no integrat). Per configurar una connexió WiFi, usar la instrucció:

Un cop oberta la connexió i transmesos / rebudes les dades és convenient tancar la connexió amb la instrucció:

Desplegables

  • Mode: Integrat o no integrat.

Entrades

  • IP Servidor: Adreça IP del servidor.
  • Port: Port de connexió.

Exemple 1

En aquest exemple s'utilitza el mòdul WiFi ESP8266 amb Arduino UNO. En l'exemple es realitzen les primeres passes de configuració per a la connexió a la xarxa i després, en el bucle principal s'envia un missatge de text a un servidor en l'adreça IP 192.168.1.100 pel port 49152 (Arduino és client de la connexió ). Si es rep alguna resposta per part del servidor, el missatge es mostra per la consola a través del port sèrie USB.
