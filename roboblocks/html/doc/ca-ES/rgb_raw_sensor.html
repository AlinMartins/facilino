Llegeix color RGB

Instrucció

Descripció

Retorna el valor en mil·lisegons del sensor RGB TCS3200 en activar un filtre: roig, verd, blau o blanc (sense filtre). La sensibilitat a cadascun dels filtres enfront d'un mateix color és diferent, podent-se utilitzar per detectar colors.

Entrades

  • Color: Color a generar
  • Pin S0: Pin S0 del sensor TCS3200.
  • Pin S1: Pin S1 del sensor TCS3200.
  • Pin S2: Pin S2 del sensor TCS3200.
  • Pin S3: Pin S3 del sensor TCS3200.
  • Pin OUT: Pin OUT del sensor TCS3200.
