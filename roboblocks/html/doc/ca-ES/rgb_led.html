Descripció

Estableix el color d'un led RGB.

Entrades

  • Pin R: Pin del LED roig.
  • Pin G: Pin del LED verd.
  • Pin B: Pin del LED blau.
  • Color: Color a generar

Exemple

En aquest exemple es canvia el color d'un LED RGB als colors roig, verd i blau seqüencialment.
