Detecta color RGB

Instrucció

Descripció

Detecta un color amb el sensor RGB TCS3200. Retorna tres possibles colors: roig, verd o blau, qualsevol altre valor retornat és un color no detectat.

Requisits

Llicència de Facilino.

Entrades

  • Pin S0: Pin S0 del sensor TCS3200.
  • Pin S1: Pin S1 del sensor TCS3200.
  • Pin S2: Pin S2 del sensor TCS3200.
  • Pin S3: Pin S3 del sensor TCS3200.
  • Pin OUT: Pin OUT del sensor TCS3200.

Eixida

Color detectat.

Exemple

En aquest exemple es mostra per consola la paraula 'ROIG', 'VERD' o 'BLAU' en funció de si es detecta el color roig, verd o blau utilitzant el sensor TCS3200. Els colors han estat prèviament calibrats.

Es perdran tots els canvis!
