Contains text string

Instruction

Description

Searches for a string of text contained into an another. If it is found, it returns true, otherwise it returns false.

Inputs

  • Contains: String of text to search for (must be of type String).
  • expression: String of text to search.

Output

True if the expression is found, false otherwise.

Example

This example looks for the expression "wo" in the phrase "Hello world!". If it is, the integrated LED on the board will turn on, otherwise it will turn it off.

All changes will be lost!
