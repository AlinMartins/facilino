Conditional function return

Instruction

Description

If the condition is true then it returns a value (leaves the function returning the indicated value). Use within the instruction:

Inputs

  • If: Evaluates whether the condition is true.
  • Return: Value to be returned.

Example

This example implements a loaded coin that has 30%chance to return 1 and 70% chance to return 0. In the example, the function is called 100 times to see how many times it has returned 1.

All changes will be lost!
