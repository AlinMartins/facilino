Color RGB

Instruction

Description

Returns an RGB color that can be compared to the result of the color detection instruction:

Requirements

Facilino's License.

Inputs

  • Color: Color to compare.

Output

Color.

Example

Example

In this example, the word 'RED', 'GREEN' or 'BLUE' is displayed on the console depending on whether the red, green or blue color is detected using the TCS3200 sensor. The colors have been previously calibrated.

All changes will be lost!
