AC Light Dimmer

Instruction

Description

This instruction generates the PWM for a thyristor to control an AC load using an AC light dimmer. This instruction uses the 'zero-crossing' signal of the AC dimmer to detect when the current of the thyristor goes to zero (must be a signal that accepts interrupt handling).

Dropdown Lists

  • Zero-Crossing PIN: The digital PIN where the zero-crossing detector is connected to.

Inputs

  • Thyristor PIN: A PWM PIN where the thyristor is connected to.
  • Brightness: AC load percentage (a value between 0 and 100).
