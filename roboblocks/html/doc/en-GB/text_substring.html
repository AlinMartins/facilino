Substring

Instruction

Description

Returns a new text string trimmed according to the specified indexes (starting from 0 for the first character).

Inputs

  • Text: String of text (must be of type String).
  • From: Index of the first character.
  • To: Index of the last character (plus one).

Output

Substring.

Example

In this example the phrase "Hello world!" is cut out, showing only "llo w".

All changes will be lost!
