Estimate altitude with barometer BMP180

Instruction

Description

Returns an estimate of the altitude with the barometric sensor BMP180 (the sensor is connected to the I2C bus). Requires that we provide atmospheric pressure at sea level (otherwise a value of 101325 pascals is provided)

Requirements

Facilino's License.

Inputs

  • Pressure: Atmospheric pressure at sea-level in Pascals.

Output

Altitude estimated by the sensor.

Example

This example shows the barometric pressure, temperature, and estimated altitude measurements with the barometric sensor BMP180.
