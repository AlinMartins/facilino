Line Tracking with TCRT5000 (Binary)

Instruction

Description

Determines the rotation speed to be applied to the base of the DYOR robot to implement a line tracking algorithm (binary). It discriminates the amount of light received to execute three different cases: Black, Gray (neither black nor white) and White.

Requirements

Facilino's License.

Inputs

  • Light: Amount of light read with the TCRT5000 sensor (analogue value).

Stacks

  • Black: Set of instructions to execute in case of detecting black.
  • Gray: Set of instructions to execute in case of not detecting black or white.
  • White: Set of instructions to execute in case of detecting white.
