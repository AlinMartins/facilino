Mathematical operations (advanced)

Instruction

Description

This instruction allows advanced mathematical operations such as square roots, absolute value (of integers), negative, natural and base10 logarithms, exponentials or base 10 powers).

Dropdown lists

  • Operation: Select the operation to perform (square root, absolute value of integers, negative, natural logarithm, logarithm in base 10, exponential or power of 10).

Inputs

Numeric input value on which to perform the operation.

Output

Value resulting from the mathematical operation. Some operations could provide NaN (Not a Number) if the operation is not defined (eg: logarithm or square root of negative numbers).

In this example, several mathematical operations are performed to demonstrate the use of the instruction.
