Read sensor temperature DS18B20

Instruction

Description

Returns the temperature in Celsius or Farenheit degrees with the temperature sensor DS18B20

Inputs

  • Bus Pin: Pin to which the sensor is connected to.

Dropdown lists

  • Units: Celsius or Farenheit.

Output

Temperature returned by the sensor.

Example

In this example the temperature read with the DS18B20 sensor is displayed on the console.

All changes will be lost!
