Bluetooth Availability

Instruction

Description

Check if bluetooth is available. If it is true then execute a set of instructions. Use in conjunction with instruction:

Requirements

Stacks

  • Available: Execute a set of instructions in case there is a data available.

Example

In this example it sends through bluetooth what is received through the serial port. In addition, everything that is received through bluetooth is sent back to the serial port.

All changes will be lost!
