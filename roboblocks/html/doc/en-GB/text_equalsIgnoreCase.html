Check if two texts are the same

Instruction

Description

Check if two text strings are the same (ignoring case).

Inputs

  • Text 1: Text 1 to compare (must be a String type variable).
  • Text 2: Text 2 to compare.

Output

True in case the text strings are the same, false otherwise.

Example

This example shows how "hello" can be compared to "HELLO" (the integrated LED on the board should flash).

All changes will be lost!
