Parse an integer on the serial port

Instruction

Description

This instruction parses an integer received on the serial port. All characters at the beginning of the text string other than digits 0 through 9 or the minus sign are ignored. Parsing ends when an invalid character is read. If no character has been received after a specified time, the function returns 0. The time can be set with the instruction:

Output

Parsed integer

Example

This example shows, through the integrated LED, the result of the selected logic operation (according to the number entered through the console). The input values ​​are determined based on the logic operation and wether the buttons are pressed.

All changes will be lost!
