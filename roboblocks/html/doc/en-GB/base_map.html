Mapping

Instruction

Description

Scale (mapping) a value between the range [0 - 1023] in another range between [0 - 'range']. It allows you to set a value (usually of the analog inputs) in another range, eg from 0 to 255.

Inputs

  • Value: value to convert.
  • Max: maximum value of the output range.

Output

Converted value.

Example

In this example the analog-to-digital conversion is displayed with a rage from 0 to 255 (originally the signal range is from 0 to 1023).

All changes will be lost!
