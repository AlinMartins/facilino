DeviceHub Open

Instruction

Description

Establishes a TCP / IP connection to DeviceHub server as client. It requires to provide the Project ID, Device UUID and API Key for publishing. To configure a WiFi connection, use the instruction:

Requirements

Facilino's license.

The ESP8266 card must be installed on the Arduino IDE Board Manager:

For ESP processors, the instruction requires to the latest version of the ESP32 software:

https://github.com/espressif/arduino-esp32

and to follow installation instructions:

https://github.com/espressif/arduino-esp32#installation-instructions

Inputs

  • Project ID: Project ID in DeviceHub.
  • Device UUID: Device UUID in DeviceHub.
  • API Key: API Key in DeviceHub.
  • Use certificate?: Uses verified SSL connection in case is selected.

Example 2

This example is a representative example for an IoT project. It reads temperature from a DHT11 temperature/humidity sensor and stores its value on a DeviceHub analog sensor. It also reads a DeviceHub digital actuator and compares its value with "1", which is equivalent to ON ("0" means OFF). After publishing data and reading data from DeviceHub the device goes for a deep sleep for 60 seconds.
