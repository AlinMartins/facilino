Do ... while / until

Instruction

Description

Run a code while or until a condition is met. The first iteration is always executed.

Stacks

  • Do: Instructions to execute

Dropdown lists

  • Mode: The code will run 'while' or 'until' the condition is met.

Inputs

  • Condition: Condition to be evaluated.
