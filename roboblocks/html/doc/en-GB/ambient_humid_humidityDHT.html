Reads humidity sensor DHT11 / DHT21 / DHT22

Instruction

Description

Returns the humidity of the air in percentage read by the sensor DHT11 / DHT21 / DHT22

Dropdown lists

  • Sensor: Model of the sensor used DHT11 / DHT21 / DHT22.

Inputs

  • Pin: Pin to which the sensor is connected to.

Output

Humidity returned by the sensor in percentage.

Example

In this example, the humidity read with the DHT11 sensor is shown on the console.

All changes will be lost!
