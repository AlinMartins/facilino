HTTP Server (WiFi ESP8266)

Instruction

Description

Defines an HTTP server using the ESP8266 WiFi module to publish HTML code. The 'integrated' variant is for the ESP8266 processor (such as NodeMCU or similar), while the 'non-integrated' variant uses AT commands by serial software to communicate with the ESP8266 WiFi module.

Requirements

Facilino's License.

The 'integrated' version requires having configured external URLs for the board management in the preferences of Arduino IDE:

Subsequently, the ESP8266 card must be installed on the Arduino IDE Board Manager:

The 'non-integrated' version requires the ITEADLIB_Arduino_WeeESP8266 library to be installed https://github.com/itead/ITEADLIB_Arduino_WeeESP8266

Dropdown lists

  • Mode: Integrated or non-integrated.

Inputs

  • SSID: Network Name (SSID).
  • Password: Password to access the network.
  • Port: Server port (default 80).
  • HTML: HTML document to publish.
  • Speed: Baud rate (maximum 38400bps).
  • TX: Arduino transmission pin (connect to RX of module ESP8266).
  • RX: Arduino receive pin (connect to TX of ESP8266 module).
