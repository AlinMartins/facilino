Radiation sensor reading (GUVA S12SD)

Instruction

Description

Returns the radiation measured value in mWatts/cm^2 or the UV-index with GUVA S12SD sensor.

Requirements

Facilino's License.

Dropdown lists

  • Radiation: Units in mWatts/cm^2 or in UV-index.

Inputs

  • Pin: Analog pin to which the sensor is connected to.

Output

Radiation level in mWatts/cm^2 or in UV-index.

Example

In this example, the values of radiation and the UV-index with GUVA S12SD sensor are displayed on the console.
