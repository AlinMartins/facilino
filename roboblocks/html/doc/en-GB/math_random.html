Generates a random number

Instruction

Description

Generates a random number (integer) between two values. The function uses a random seed based on the reading of analogue channel A0 (a typical trick to emulated randomness).

Inputs

  • Min: Initial value of the range.
  • Max: End value of the range.

Output

Random number generated.

Example

In this example 100 random numbers are generated.

All changes will be lost!
