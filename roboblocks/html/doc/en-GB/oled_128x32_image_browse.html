Provides custom image (OLED 128x32 I2C)

Instruction

Description

Provides a custom image (width must be multiple of 8). This instruction transforms a black and white image (or a grayscaled image) into image data to be drawn in a OLED 128x32 I2C device. Nota that once the image is selected, it will be processed and internally coded in the instruction, so if the original image changes, the data generated with the instruction won't change. This allows you to select it once, save your Facilino code for later use without needing to select again the same file every time. Text Input Fields

  • Image name: Image name. To avoid duplicity of copies of the same image, you should use the same name..

Buttons

  • Browse: Input button to select the image file.
