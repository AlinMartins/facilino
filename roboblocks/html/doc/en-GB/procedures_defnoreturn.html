Non-return function (definition)

Instruction

Description

Implements the call of a function that has no return value. The code to execute in the function must be stacked inside the statement. The function can have multiple input arguments (and of different types) using the mutator (top left icon). Use in combination of instruction:

Statement Instructions

  • Execute: Instructions to execute within the function.

Mutator

Allows you to create input parameters of the function. Add as many variables as needed, specify their name and type of variable.

Elements

Inputs
