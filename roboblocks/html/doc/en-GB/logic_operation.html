Logical operators

Instruction

Description

It evaluates the inputs and returns true or false depending on the selected logic operator (AND, OR, XOR, XNOR or LOGICAL IMPLICATION).

Dropdown lists

  • Operator: Logical operator to evaluate.

Inputs

  • A: Logical value to be evaluated.
  • B: Logical value to be evaluated.

Output

Evaluated binary value (true or false).

Example

This example shows, through the integrated LED, the result of the selected logic operation (according to the number entered through the console). The input values ​​are determined based on the logic operation and wether the buttons are pressed.
