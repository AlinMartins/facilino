RGB Led

Instruction

Description

Sets the color of an RGB led.

Inputs

  • Pin R: Red LED Pin.
  • Pin G: Green LED Pin.
  • Pin B: Blue LED Pin.
  • Color: Color to generate

Example

In this example, the color of an RGB LED is changed to the red, green and blue colors sequentially.

All changes will be lost!
