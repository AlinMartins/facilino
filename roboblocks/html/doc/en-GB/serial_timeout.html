Set Timeout (serial port)

Instruction

Description

Sets the waiting time for reading data on the serial port. By default the value is 1000ms.

Inputs

  • Time: Waiting time in milliseconds.

Example

This example waits for the letter 's' to begin with the rest of the program. In the ASCII table, the letter 's' corresponds to the value 115.

All changes will be lost!
