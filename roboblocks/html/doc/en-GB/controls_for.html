Repeat ... (count)

Instruction

Description

Repeat a set of instructions a number of times. It uses a variable to iterate from an initial value to an end value, the number of times the instructions are repeated is the difference between the final value minus the initial value plus one.

Statement Instructions

  • do: Set of instructions to run repetitively

Inputs

  • with: variable to use in iterations (iterator).
  • from: initial value that the iterator takes
  • to: final value reached by the iterator
