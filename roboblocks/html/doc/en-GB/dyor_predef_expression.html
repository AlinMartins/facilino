Generates predefined expression in LEDs matrix

Instruction

Description

It allows to generate a predefined expression for the robot DYOR with the LEDs matrix (max7219). It allows to select between five predefined expressions, as well as the orientation of the LEDs matrix.

Requirements

Facilino's License.

Inputs

  • CS: Pin CS of the LED array (max7219).
  • DIN: Pin DIN of the matrix of LEDs (max7219).
  • CLK: Pin CLK of the LED array (max7219).

Dropdown lists

  • Orientation: Vertical, Horizontal, Vertical 180º or Horizontal 180º
  • Expression: Angry, Happy, Sad, In Love, Surprised.

Example

In this example, the DYOR robot generates the expression 'Happy' and 'Angry' alternatively with waiting times of 1000 milliseconds.

All changes will be lost!
