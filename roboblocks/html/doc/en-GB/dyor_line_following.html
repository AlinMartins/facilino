Line Tracking with TCRT5000 (Proportional)

Instruction

Description

Determines the rotation speed to be applied to the base of the DYOR robot to implement the line tracking algorithm. The correction rotational speed is proportional to the difference between the read light level and an intermediate reference light value (between black and white).

Requirements

Facilino's License.

Inputs

  • Light: Amount of light read with the TCRT5000 sensor (analogue value).
  • Gain: Correction factor, the higher gain the faster correction movements. If it is too low, it may not be correct.

Output

Returns the proportional spin value (between -100% and 100%).
