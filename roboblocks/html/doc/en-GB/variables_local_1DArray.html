Declares an array

Instruction

Description

Declares an array using an array constructor. The size of the array is taken from the size of the array constructor. It can also copy the values of an existing array variable, taking the same size and variable type as the input variable.

Fields

  • Name: Array name.

Inputs

  • Value: Value to assign to the array.

Example

This example shows how to use array variables and constructor. First, it declares a global and local array and then it creates a copy of the local array. The elements of the new created array are then modified with values of the global array. The resulting operation is shown on the console.

All changes will be lost!
