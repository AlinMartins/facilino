Declares a global array with type

Instruction

Description

Declares a global array using an array constructor specifying its type. The size of the array is taken from the size of the array constructor. It can also copy the values of an existing array variable, taking the same size and variable type as the input variable.

Fields

  • Name: Array name.
  • Type: Select variable type 'String', 'char', 'interger', 'long integer', 'byte', 'boolean' or 'float'.

Inputs

  • Value: Value to assign to the array.
