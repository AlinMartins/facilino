Wait to meet a condition

Instruction

Description

Wait until a condition between two values ​has been met.

Inputs

  • Value1: Value 1 to be evaluated.
  • Value2: Value 2 to be evaluated.

Example

In this example the program waits for a button to be pressed connected to pin 2. After pressing it, the integrated LED on the board lights up.

All changes will be lost!
