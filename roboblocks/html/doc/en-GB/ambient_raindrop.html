Rain sensor (analog reading)

Instruction

Description

Returns the value of the rain sensor's analog signal.

Inputs

  • Pin: Analog pin to which the sensor is connected to.

Output

Analog reading (from 0 to 1023).

Example

In this example, the analogue reading of the rain sensor is shown on the console and in case its digital output is activated because it rains, shows a rain warning message.

All changes will be lost!
