Array (with four numbers) constructor

Instruction

Description

Defines an array constant with two numbers.

Inputs

  • Element 1: First element of the array.
  • Element 2: Second element of the array.
  • Element 3: Third element of the array.
  • Element 4: Fourth element of the array.

Output

Constant array.

Example

This example shows how to use constant dimensional array variables. First, it declares a global and local array and then it creates another local array. The elements of the new created array are assigned with values of the global and local arrays. The resulting values are shown on the console.

All changes will be lost!
