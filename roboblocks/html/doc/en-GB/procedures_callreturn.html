Function with return (call)

Instruction

Description

Performs a call to a function that has a return value returned at the output connector. If the function has input arguments, these can be passed to the function with the input connectors of the instruction itself. Use in combination of instruction:

Dropdown lists

  • Function: Name of the function to call.

Output

Value returned by the function.
