Read RGB color

Instruction

Description

Returns the value in milliseconds of the TCS3200 RGB sensor when a filter is activated: red, green, blue or white (without filter). The sensitivity to each of the filters against the same color is different, and can be used to detect colors.

Inputs

  • Color: Color to generate
  • Pin S0: Pin S0 of sensor TCS3200.
  • Pin S1: Pin S1 of the TCS3200 sensor.
  • Pin S2: Pin S2 of the TCS3200 sensor.
  • Pin S3: Pin S3 of TCS3200 sensor.
  • Pin OUT: Pin OUT of TCS3200 sensor.
