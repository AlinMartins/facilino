Multi-tareas (periódicas)

Instrucción

Descripción

Permite definir un conjunto de tareas periódicas utilizando el temporizador 0 de Arduino. Todas aquellas instrucciones que utilizan el temporizador 0 o retardos pueden verse afectadas. Dispone de una parte de código que siempre se ejecuta cada milisegundo y otra parte que se ejecuta periódicamente según se crean nuevas tareas con el mutador.

Instrucciones apilables

  • Ejecutar siempre: Conjunto de instrucciones a ejecutar cada milisegundo.
  • Tarea: Conjunto de instrucciones a ejecutar al periodo especificado.

Entradas

  • Periodo: Periodo de la tarea a ejecutar en milisegundos.
