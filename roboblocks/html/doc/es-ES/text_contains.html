Continene cadena de texto

Instrucción

Descripción

Busca una cadena de texto contenida en otra. Si se encuentra, devuelve verdadero, de lo contrario, devuelve falso.

Entradas

  • Contiene: Cadena de texto donde buscar (debe ser del tipo cadena de texto).
  • expressión: Cadena de texto a buscar.

Salida

Verdadero si se encuentra la expressión, falso en caso contrario.

Ejemplo

En este ejemplo se busca la expresión "wo" en la frase "Hello world!". Si se encuentra enciende el LED integrado en la placa, de lo contrario, lo apaga.

Se perderán todos los cambios!
