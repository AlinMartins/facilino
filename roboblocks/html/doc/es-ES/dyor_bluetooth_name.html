Configura nombre bluetooth

Instrucción

Descripción

Configura el nombre del módulo bluetooth HC-06 y similares mediante comunicación serie por software con comandos AT. El cambio del nombre es permanente, quiere decir que una vez cambiado se mantiene. Requiere de la velocidad de comunicación y los pines a los que está conectado. En Arduino UNO y similares, los pines 0 y 1 se utilizan para la comunicación serie hardware, mientras que esta instrucción permite utilizar el módulo bluetooth HC-06 y similares mediante comunicación serie por software y por tanto la conexión de los pines de transmisión y recepción deben especificarse, así como la velocidad de comunicación (por defecto 9600 baudios).

Requisitos

Licencia de Facilino

Entradas

  • Velocidad: Velcidad de transmisión en baudios. Valor por defecto 9600. Nota: por encima de 38400 no suele funcionar bien.
  • TX: Pin de transmisión de Arduino (conectar a RX del módulo HC-06).
  • RX: Pin de recepción de Arduino (conectar a TX del módulo HC-06).
  • Nombre: Nombre a configurar (el dispositivo bluetooth debe aparecer con ese nombre al cabo de un tiempo).

Ejemplo

En este ejemplo se modifica el nombre que tiene por defecto el módulo bluetooth por el de "ROBOTICAFACIL".
