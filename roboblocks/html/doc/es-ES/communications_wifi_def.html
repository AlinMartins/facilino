Define WiFi ESP8266

Instrucción

Descripción

Define el módulo WiFi ESP8266. La variante 'integrado' es para el procesador ESP8266 (como NodeMCU o similares), mientras que la variante 'no integrado' utiliza comandos AT por serie software para comunicar con el módulo WiFi ESP8266.

Requisitos

La versión 'integrado' requiere tener configurado URLs externas para la gestión de tarjetas (board manager) en las preferencias de Arduino IDE:

Posteriormente, se debe instalar la tarjeta ESP8266 en el Board Manager de Arduino IDE.:

La versión 'no integrado' requiere tener instalada la librería ITEADLIB_Arduino_WeeESP8266 https://github.com/itead/ITEADLIB_Arduino_WeeESP8266

Desplegables

  • Modo: Integrado o no integrado.

Entradas

  • SSID: Nombre de red (SSID).
  • Contraseña: Contraseña de acceso a la red.
  • Velocidad: Tasa de baudios (máximo 38400bps).
  • TX: Pin de transmisión de Arduino (conectar a RX del módulo ESP8266).
  • RX: Pin de recepción de Arduino (conectar a TX del módulo ESP8266).
