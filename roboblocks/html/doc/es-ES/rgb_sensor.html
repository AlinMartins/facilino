Detecta color RGB

Instrucción

Descripción

Detecta un color con el sensor RGB TCS3200. Devuelve tres posibles colores: rojo, verde o azul, cualquier otro valor devuelto es un color no detectado.

Requisitos

Licencia de Facilino.

Entradas

  • Pin S0: Pin S0 del sensor TCS3200.
  • Pin S1: Pin S1 del sensor TCS3200.
  • Pin S2: Pin S2 del sensor TCS3200.
  • Pin S3: Pin S3 del sensor TCS3200.
  • Pin OUT: Pin OUT del sensor TCS3200.

Salida

Color detectado.

Ejemplo

En este ejemplo se muestra por consola la palabra 'ROJO', 'VERDE' o 'AZUL' en función de si se detecta el color rojo, verde o azul utilizando el sensor TCS3200. Los colores han sido previamente calibrados.
