Recibe comando

Instrucción

Descripción

Recibe comandos por bluetooth de una aplicación externa. Ejecuta un conjunto de instrucciones en función del comando recibido. Dispone de una opción para repetir un comando, quiere decir que una vez enviado, éste se recuerda internamente hasta que se recibe el comando '0' (cancelar repetición).

Requisitos

Licencia de Facilino

Versiones anteriores
En versiones anteriores de Facilino, esta instrucción disponía de los pines TX y RX configurables. Esta opción se ha eliminado para simplificar es aspecto de la instrucción. Debes asegurarte que usas la instrucción de definición del módulo bluetooth para el correcto funcionamiento.

Elementos

Entradas

  • Comando: Número del comando a ejecutar.
  • Repite: Si está marcado, recuerda el comando para repetir las instrucciónes hasta que se envía el comando 0.

Instrucciones apilables

  • Hacer: Conjunto de instrucciones a ejecutar en caso de recibir el comando.

Ejemplo 2

En este ejemplo se leen carácteres ASCII por bluetooth. Si el usuario envía un '0' (30 en ASCII), se encienden dos LEDs conectados a los pines 5 y 6. Si el usuario envía un '1' (31 en ASCII), se enciende el LED conectado al pin 6, mientras que si se envía un '2' (32 en ASCII), se enciende el LED conectado al pin 5. El comando '3' (33 en ASCII) desactivará los LEDs.

Se perderán todos los cambios!
