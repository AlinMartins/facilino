Lee color RGB

Instrucción

Descripción

Devuelve el valor en milisegundos del sensor RGB TCS3200 al activar un filtro: rojo, verde, azul o blanco (sin filtro). La sensiblidad a cada uno de los filtros frente a un mismo color es diferente, pudiéndose utilizar para detectar colores.

Entradas

  • Color: Color a generar
  • Pin S0: Pin S0 del sensor TCS3200.
  • Pin S1: Pin S1 del sensor TCS3200.
  • Pin S2: Pin S2 del sensor TCS3200.
  • Pin S3: Pin S3 del sensor TCS3200.
  • Pin OUT: Pin OUT del sensor TCS3200.
