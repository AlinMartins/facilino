Sensor de infrarrojo (TCRT5000)

Instrucción

Descripción

Obtiene la cantidad de luz recibida por el sensor infrarrojo TCRT5000 utilizando la salida analógica (AO).

Entradas

  • Pin: Pin de la señal analógica (AO) del sensor TCRT5000.

Salida

Devuelve el valor analógico de la cantidad de luz (próximos a 0 con blanco puro y próximos a 1023 para negro puro).
