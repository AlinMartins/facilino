Devuelve un comando con el valor indicado. Usar junto con la instrucción:

Requisitos

Licencia de Facilino

Entradas

  • Valor: valor del comando.

Salida

Comando indicado.

Ejemplo

En este ejemplo se leen carácteres ASCII por bluetooth. Si el usuario envía un '0' (30 en ASCII), se encienden dos LEDs conectados a los pines 5 y 6. Si el usuario envía un '1' (31 en ASCII), se enciende el LED conectado al pin 6, mientras que si se envía un '2' (32 en ASCII), se enciende el LED conectado al pin 5. El comando '3' (33 en ASCII) desactivará los LEDs.

Se perderán todos los cambios!
