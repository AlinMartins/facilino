LCD (2x16)

Instrucción

Descripción

Define las conexiones de una pantalla LCD 2x16 basado en el chip Hitachi HD44780 (o compatible). Utiliza el modo de trasmisión de 4-bits de datos (de D4 a D7).

Entradas (según orden y nomenclatura de los pines de la pantalla LCD)
  • RS: Selección de registro.
  • E: Señal de habilitación de la pantalla.
  • D4: Bit 4 de datos
  • D5: Bit 5 de datos
  • D6: Bit 6 de datos
  • D7: Bit 7 de datos

Requisitos

Requiere tener instalada la librería 'LiquidCristal' en Arduino IDE.

Ejemplo

En este ejemplo se muestra el Circuito de la pantalla LCD a la tarjeta de expansión de Arduino Nano. En el código del ejemplo se muestra la frase 'Hello world!" en la pantalla LCD.
