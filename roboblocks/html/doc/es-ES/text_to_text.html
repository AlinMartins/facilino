Convierte a cadena de texto

Instrucción

Descripción

Convierte la información de entrada a texto o caracter.

Desplegables

  • Conversión: Determina el tipo de conversión a realizar.

Entradas

  • Entrada: Información a convertir.

Salida

Cadena de texto o caracter convertido.

Ejemplo

En este ejemplo por un lado se convierte el número 12345 a una cadena de texto para determinar el número de dígitos. Por otro lado, se convierte el número 113 que corresponde a la letra 'q' en ASCII.

Se perderán todos los cambios!
