Mientras/Hasta que...

Instrucción

Descripción

Ejecuta un código mientras/hasta que se cumpla una condición. Evalúa la condición antes de ejecutar la primera iteración.

Instrucciones apilables

  • Instrucciones a ejecutar

Desplegables

  • Modo: El código se ejecutará 'mientras' o 'hasta que' se cumpla la condición.

Ejemplo 1

En este ejemplo, el programa se espera mientras el usuario no apriete el botón (estado alto).

Se perderán todos los cambios!
