Busca cadena de texto

Instrucción

Descripción

Devuelve la posición de la primera o última instancia de una cadena de texto a buscar. Si no se encuentra, devuelve -1.

Desplegables

  • Instancia: Primera o última instancia

Entradas

  • Busca: Cadena de texto a buscar.
  • en: Cadena de texto donde buscar (debe ser del tipo cadena de texto).

Salida

Posición del caracter de la primera instancia o última instancia encontrada.

Ejemplo

En este ejemplo se busca la letra "o" en la frase "Hola mundo!". La primera instancia está en la posición 2 y la última instancia en la posición 9.

Se perderán todos los cambios!
