Recorta cadena de texto

Instrucción

Descripción

Devuelve una nueva cadena de texto recortada según los índices especificados (empezando desde 0 para el primer caracter).

Entradas

  • Texto: Cadena de texto (debe ser del tipo cadena de texto).
  • Desde: Índice del primer caracter.
  • Hasta: Índice del último caracter (más uno).

Salida

Cadena de texto recortada.

Ejemplo

En este ejemplo se recorta la frase "Hola mundo!", mostrándose sólo "la mu".

Se perderán todos los cambios!

Código Facilino
