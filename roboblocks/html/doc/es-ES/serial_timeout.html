Tiempo de espera (puerto serie)

Instrucción

Descripción

Establece el tiempo de espera para la lectura de un dato por el puerto serie. Por defecto el valor es de 1000ms.

Entradas

  • Tiempo: Tiempo de espera en milisegundos.

Ejemplo

En este ejemplo se espera a que se pulse la letra 's' para empezar con el resto del programa. En la tabla ASCII, la letra 's' corresponde al valor 115.

Se perderán todos los cambios!
