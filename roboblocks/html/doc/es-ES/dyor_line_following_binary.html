Seguimiento de líneas con TCRT5000 (Binario)

Instrucción

Descripción

Determina la velocidad de giro a aplicar a la base del robot DYOR para implementar el algoritmo de seguimiento de líneas binario. Discrimina la cantidad de luz recibida para ejecutar tres diferentes casos: Negro, Gris (ni negro ni blanco) y Blanco.

Requisitos

Licencia de Facilino.

Entradas

  • Luz: Cantidad de luz leída con el sensor TCRT5000 (valor analógico).

Instrucciones apilables

  • Negro: Conjunto de instrucciones a ejecutar en caso de detectar negro.
  • Gris: Conjunto de instrucciones a ejecutar en caso de no detectar negro ni tampoco blanco.
  • Blanco: Conjunto de instrucciones a ejecutar en caso de detectar blanco.
