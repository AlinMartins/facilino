Atributo de etiqueta HTML

Instrucción

Descripción

Crea un atributo para una etiqueta HTML.

Desplegables

  • Atributo: Tipo de atributo a crear.
  • Código: Código para el atributo.

Salida

Código HTML para insertar en los atributos de una etiqueta HTML.

Ejemplo

En este ejemplo se utiliza el procesador ESP8266 (integrado) para crear una web que muestra un título en rojo y el texto con formato "Hola mundo!"
