Operaciones matemáticas (avanzadas)

Variantes de la instrucción

Descripción

Esta instrucción permite realizar operaciones matemáticas avanzadas como raíces cuadradas, valor absoluto (de números enteros), negativo, logaritmos neperiano y en base 10, exponenciales o potencias de base 10).

Desplegables

  • Operación: Selecciona la operación a realizar (raíz cuadrada, valor absoluto de números enteros, negativo, logaritmo neperiano, logaritmo en base 10, exponencial o potencia de 10).

Entradas

Valor numérico de entrada sobre el cual realizar la operación.

Salida

Valor resultante de la operación. Algunas operaciones podrían proporcionar NaN (Not a Number) si la operación no está definida (p.e.: logaritmo o raíz cuadrada de números negativos).
