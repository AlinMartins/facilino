Mapear

Instrucción

Descripción

Escala (mapeo) un valor comprendido entre el rango [0 - 1023] en otro rango comprendido entre [0 - 'rango']. Permite ajustar un valor (normalmente de las entradas analógicas) en otro rango, p.e.: de 0 a 255.

Entradas

  • Valor: valor a convertir.
  • Max: valor máximo del rango de salida.

Salida

Valor convertido.
