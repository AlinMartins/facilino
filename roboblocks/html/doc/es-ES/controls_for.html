Repetir...(un número de veces)

Instrucción

Descripción

Repite un conjunto de instrucciones un número de veces. Utiliza una variable para iterar desde un valor inicial hasta un valor final, siendo el número de veces que se repiten las instrucciones la diferencia entre el valor final menos el valor inicial más uno.

Instrucciones apilables

  • ejecutar: Conjunto de instrucciones a ejecutar de forma repetitiva

Entradas

  • con: variable para utilizar en las iteraciones (iterador).
  • desde: valor inicial que toma el iterador
  • hasta: valor final que alcanza el iterador
