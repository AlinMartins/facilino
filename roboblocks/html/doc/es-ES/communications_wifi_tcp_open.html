Abre conexión TCP/IP Cliente (ESP8266)

Instrucción

Descripción

Establece una conexión TCP/IP con un servidor como cliente TCP/IP. Requiere de la IP del servidor y el puerto a comunicar y seleccionar el modo (WiFi integrado o no integrado). Para configurar una conexión WiFi, usar la instrucción:

Una vez abierta la conexión y transmitidos/recibidos los datos es conveniente cerrar la conexión con la instrucción:

Desplegables

  • Modo: Integrado o no integrado.

Entradas

  • IP Servidor: Dirección IP del servidor.
  • Puerto: Puerto de conexión.

Ejemplo 1

En este ejemplo se utiliza el módulo WiFi ESP8266 con Arduino UNO. En el ejemplo se realizan los primeros pasos de configuración para la conexión a la red y después, en el bucle principal se envía un mensaje de texto a un servidor en la dirección IP 192.168.1.100 por el puerto 49152 (Arduino es cliente de la conexión). Si se recibe alguna respuesta por parte del servidor, el mensaje se muestra por la consola a través del puerto serie USB.
