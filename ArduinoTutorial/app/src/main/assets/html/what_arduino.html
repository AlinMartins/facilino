What's Arduino?

Let's get started and learn about Arduino

Competencies

  • Acquire interest in new Technologies.
  • Introduce Arduino.

Are you interested in learning about Arduino? So why don't get started right now and discover an amazing set of features with Arduino.

Due to its open-hardware design, Arduino has become one of the most used electronics widely used by many people, from beginners to professionals. All Arduino boards are powered by ATMEL microcontrollers. A microcontroller is the brain of the board and will allow you upload a code that will be executed on power-up. With your code, you will be able to interact with things in the real world, such as to read information from sensors, move motors, light on LEDs, print information over the console and many more... 

Related with Arduino, you can find a set of electronics ready-to-use with very different types of sensors and actuators. The ones that are connected on top of Arduino's board are known as shields, and they are, probably, the best way to get started with. They give you access to all Arduino's features through a selected group of sensors and actuators.

The capabilities of the Arduino are almost limitless. In this tutorial we will cover everything you need to know to getting started. It will help you to learn the fundamental skills and knowledge that you need to get your Arduino up and running. It should serve as a basis to introduce you into further experimentation and learning.

Programming electronic devices has never been so easy. Arduino uses Arduino IDE, a text programming tool with many examples. This tutorial uses Facilino, a block programming tool for Arduino. If you have no experience with Arduino this tutorial will help to getting started while at the same time you learn programming with Facilino.
