Senoide

Instruction

Description

This instruction allows to generate sine waves with definable amplitude, frequency, phase and mean value. The resulting value depends on the evaluated time.

Chart

Inputs

  • Amplitude: amplitude of the signal (difference between peak and peak).
  • Frequency: Number of times the signal is repeated per second (Hz).
  • Phase: Horizontal shift of the signal (angle in degrees).
  • Mean value: Vertical displacement of the signal.
  • Time: The instant of time in which the signal is evaluated (in microseconds).

Output

Resultant value of the signal.

Example

In this example an oscillating motion is performed on a position servo (these movements are those used in some walking robots).

All changes will be lost!
