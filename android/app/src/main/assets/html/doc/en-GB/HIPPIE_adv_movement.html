Generates a movement for DYOR bPED Robot

Instruction

Description

This instruction generates different movements with configurable parameters.

Use in combination with the following instruction:

Requirements

Facilino license

Dropdown lists

  • Movement: Select the movement type.

Inputs

  • Steps: Number of steps (or repetitions) of the movement.
  • Speed: Set 0% for minimum speed and 100% for maximum speed.
  • Configuration: Set 0% for a minimum angle configuration and 100% for a maximum angle configuration.

Example

This example generates all possible predefined movements for DYOR bPED robot.

All changes will be lost!
