Draws a stream of text in the LEDs matrix

Instruction

Description

This instruction accepts a stream of numbers (corresponding to a stream of text) to be shown in the 8x8 LEDs matrix (max7219). Given a delay time, the text will be shifted to the left (one column at each step) to show to complete text stream. Use in combination with the following instruction:

Requirements

Facilino's License.

Inputs

  • CS: Pin CS of the LED array (max7219).
  • DIN: Pin DIN of the matrix of LEDs (max7219).
  • CLK: Pin CLK of the LED array (max7219).
  • Stream text: Text to show in a stream.
  • Wait: Transition wait time between consecutive streams.

Dropdown list

  • Configuration: Configuration of the LEDs matrix (vertical, horizontal, vertical 180º or horizontal 180º).

Example

In this example, we show the expression "DYOR is cool!" in the LEDs matrix with a controlled delay of 100ms on every shift.
