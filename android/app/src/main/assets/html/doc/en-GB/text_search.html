Search for text string

Instruction

Description

Returns the position of the first or last instance of a string to search. If it is not found, it returns -1.

Dropdown lists

  • Instance: First or last instance

Inputs

  • Search: Text string to search.
  • in: String of text where to search (must be of type String).

Output

Position of the character of the first instance or last instance found.

Example

In this example the letter "o" is searched for in the phrase "Hello world!". The first instance is in position 4 and the last instance in position 7.

All changes will be lost!
